## Processo de criação

O TCC retratava o tema de "Planejamento Financeiro Familiar", para desenvolver um site de forma rápida e eficiente, foi utilizado o React TS.

- Dashboard: Consiste em gráficos que conectam com o DB local, mostrando facilmente o status do cliente em relação a suas transações
- Transações: Possui uma tabela mais intuitiva, com botões funcionais de criar, filtrar, ou até mesmo deletar transações, alterando os gráficos do Dashboard
